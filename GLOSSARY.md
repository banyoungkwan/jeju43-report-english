# Master Glossary / 용어집

> This glossary defines the key terms, place names, and romanization conventions used consistently throughout the English translation of the *Jeju 4·3 Incident Truth Investigation Report* (2003).
> 이 용어집은 영문 번역 전반에 걸쳐 일관되게 사용된 주요 용어, 지명, 로마자 표기 기준을 정리한 것입니다.

## Part 4. Romanization Conventions / 로마자 표기 원칙

This translation uses **McCune-Reischauer (MR) romanization** for Korean proper nouns, with the following exceptions and conventions:
